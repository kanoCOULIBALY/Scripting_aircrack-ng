Usage: script.py [OPTION] [ARGUMENTS]

Options:
  -ms, --monitoring_scaning <interface>     Enable monitoring mode and perform scanning.
                                 Arguments: <interface>

  -m, --monitoring <interface>              Enable monitoring mode only.
                                 Arguments: <interface>

  -s, --scan <interface>                    Perform scanning only.
                                 Arguments: <interface>

  -h, --help                                Display this help text.

  -wpa                                     WPA-specific operations.
    -ha, --handshake <channel> <BSSID> <interface>             Capture handshake packets.
                                 Arguments: <interface> <BSSID> <channel>
    -d, --deathauth <interface> <BSSID> <client> <channel>            Perform deauthentication attack.
                                 Arguments: <interface> <BSSID> <client> <channel>
    -c, --cracking               Crack WPA/WPA2 key using a handshake file.
                                 Arguments: <handshake_file> <wordlist>

  -wep                                     Placeholder for WEP operations.
  -wps                                     Placeholder for WPS operations.
